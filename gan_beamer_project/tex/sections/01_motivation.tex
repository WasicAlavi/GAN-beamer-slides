\section{Why We Need GANs}

\begin{frame}{Why Generative Models?}
\begin{itemize}
\item Discriminative models classify data.
\item Generative models create new data.
\item Applications: image synthesis, data augmentation, creative AI.
\end{itemize}
\end{frame}

\begin{frame}{Why Is It Hard?}
\begin{itemize}
\item Complex high-dimensional distributions.
\item Likelihood often intractable.
\item Traditional methods require complex sampling.
\end{itemize}
\end{frame}
